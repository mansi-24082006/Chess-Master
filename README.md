# Chess-Master

A complete Chess game built using **HTML, CSS, and JavaScript**, featuring standard chess rules, interactive gameplay, and a clean user interface.  
This project demonstrates strong JavaScript logic handling for piece movement, turn management, and game state control.

---

## 🚀 Features

- ♟️ Full chessboard with standard layout
- 👑 All chess pieces with correct movement logic
- 🔄 Turn-based gameplay (White vs Black)
- ❌ Invalid move prevention
- 🎯 Piece capture logic
- 🔁 Game state updates in real-time
- 🎨 Responsive and clean UI using CSS
- ⚡ Built with pure JavaScript (no libraries)

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the chessboard
- **CSS3** – Styling and layout
- **JavaScript (Vanilla JS)** – Game logic and interactions
